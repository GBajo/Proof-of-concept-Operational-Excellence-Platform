Cómo usar

1. Añadir documentos

Copia tus archivos PDF, DOCX o XLSX a la carpeta docs/ del servidor. Formatos soportados: PDF, Word, Excel.

Pulsa "Re-indexar todos" o usa el botón individual por archivo. También puedes ejecutar python ingest.py desde terminal.

3. Asistente RAG

Los fragmentos indexados se usan automáticamente como contexto cuando el operario registra un comentario o pide ayuda al asistente.
